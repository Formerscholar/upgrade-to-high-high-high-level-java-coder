## TS的简介与安装



#### 一、TypeScript的简介

是在javascript基础上的一层封装，使得TypeScript更加面向对象编程。TypeScript都会被编译成javascript执行。





#### 二、安装TypeScript

需要先安装好nodejs的环境，保证npm有淘宝的镜像加速。



- 使用如下命令安装TypeScript

  ```typescript
  npm install -g typescript
  ```

- 运行如下命令，查看安装的typescript的版本

  ```shell
  tsc -v
  ```
